Introduction:
-------------
The Student Report Card Management System in C++ is a desktop application coded in a C++ programming language. The project contains basic functionality that allow user to generate the student report card. This project was created to fasten the distribution of grade record to each student. This will make it easier for the teacher to compute the General Average for the student grade.

Basic Information:
------------------
Language Used:- C++
Coding tool Used:- VS Code and Code Blocks
Type: Desktop Application
Database Used: .dat file extension generated using C++ file system.

Features:
---------
• Can Add Student Information

• Display Student Individual Grade

• Search Existing Student

• Update Student Grade
